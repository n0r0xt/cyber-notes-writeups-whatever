# narnia 1 writeup
## this is my writeup for the narnia 1 wargame on overthewire

- access via ssh as narnia1@narnia.labs.overthewire.org on port 2226 with the password from the last level
- you are given 2 files: narnia1 binary and narnia1.c source code
- the environment comes with tools like r2 and gdb to use
- password for the next level is at /etc/narnia_pass/narnia2
